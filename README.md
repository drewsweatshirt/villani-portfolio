This is my first fully coded website, I'm just gonna make some notes so I can remember how to do this process in case I gotta again on some nerd shit

1. Purchase a domain (namecheap, google domains, etc)
2. Get Github, Git, VS Code, Vercel
3. Make a folder containing the core components to a website:
       a. index.html
       b. style.css
       c. script.js
       d. /images
4. Create repository on Github: name it "<username>.github.io"
5. Add index.html
6. Go back to Namecheap and manage your domain
       a. Advanced DNS > Host Records > Add New:
                A Record → @ → 185.199.108.153
                A Record → @ → 185.199.109.153
                A Record → @ → 185.199.110.153
                A Record → @ → 185.199.111.153
                CNAME Record → www → <username>.github.io
                *** ALL TTL: Automatic ***
7. Create a new file inside repository:
        a. Name: "CNAME"
        b. Line 1: "<domain.address>"
